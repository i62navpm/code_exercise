# Cabify Coding Exercise
Cabify code excercise

## Installation
1. Clone repository
  ```bash
  git clone https://github.com/i62navpm/code_exercise.git
  ```

2. Install yarn
  ```bash
  npm install --global yarn
  ```

3. Install dependencies
  ```bash
  yarn install
  ```

4. Build script
  ```bash
  yarn build
  ```

5. 
  a. Run tests 
    ```bash
    yarn test
    ```
  b. Run script 
    ```bash
    yarn run-build
    ```
